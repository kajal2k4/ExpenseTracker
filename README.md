## ✨ Features
- 🔐 User auth (signup/login + password recovery)
- 📊 Interactive donut chart (Chart.js)
- ➕ Add/delete transactions
- 🔍 Filter by category/date
- 📱 Fully responsive

---

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **UI**: Bootstrap 5, Font Awesome
- **Charts**: Chart.js
- **Storage**: localStorage
